     to-transparent (bottom half only)

     Inactive: w-2 h-2 bg-white/40 rounded-full

     transition with Framer Motion layout)
     Clickable to jump to slide
   - ARROWS (Desktop only, hidden on mobile):
     Left/Right chevron icons (ChevronLeft, ChevronRight from lucide-react)
     Positioned absolute, vertically centered, inset-x edges (left-4, right-4)

     Opacity 0 by default, opacity 100 on carousel hover (group-hover)
     Framer Motion: whileHover scale 1.1, whileTap scale 0.95

   - Touch swipe: Use onTouchStart/onTouchEnd to detect horizontal swipes
     (threshold: 50px). Swipe left → next, swipe right → prev
   - Keyboard: When carousel is in viewport, left/right arrow keys navigate
     (add event listener, remove on unmount/out of viewport)
   - Prevent vertical scroll interference on touch swipe

   - Use Framer Motion AnimatePresence with mode="wait"
   - Slide enters from right (x: 300, opacity: 0) → (x: 0, opacity: 1)
   - Slide exits to left (x: -300, opacity: 0)
   - When going backward: reverse direction (enter from left, exit to right)
   - Duration: 0.5s, ease: [0.25, 0.1, 0.25, 1] (custom cubic-bezier)

     Show a skeleton placeholder: same aspect ratio, bg-gray-200,
     animate-pulse, with a centered faded PakCart logo watermark
   - If no slides exist or fetch fails: Show a static fallback banner
     with brand colors (green gradient) and "Welcome to PakCart" text

8. PERFORMANCE:
   - Preload next slide's image using <link rel="preload"> in head
     (use react-helmet-async) or new Image() in JS
   - First slide image: priority loading, fetchpriority="high"
   - Use will-change: transform on active slide for GPU acceleration
   - Cloudinary transformations: f_auto (format), q_auto (quality),
     w_1920 (desktop) / w_768 (mobile), c_fill
